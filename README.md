
🚀 CampusMind AI – Smart Campus Management Platform
📌 Overview

CampusMind AI is a full-stack SaaS platform for managing campus issues with AI-powered features and multi-tenant architecture.

🧱 Project Structure
campusmind-ai/
│
├── backend/    # FastAPI backend
├── frontend/   # React frontend
├── README.md

🚀 Features
Multi-tenant system (college-based)
JWT Authentication + Role-based access
Issue reporting with image upload
Category management (college-specific)
AI Chatbot (Gemini)
AI Study Planner
Dashboard with filters & pagination


🛠️ Tech Stack
Backend: FastAPI, SQLAlchemy
Frontend: React.js, Axios
Database: SQLite
AI: Google Gemini


⚡ Quick Start
Run Backend
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8001

Run Frontend
cd frontend
npm install
npm start
🌐 URLs
Frontend → http://localhost:3000
Backend → http://127.0.0.1:8001/docs
